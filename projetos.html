JO-KEN-PO

Minha última aplicação para web, com a minha bagagem de conhecimento mais recente me permitiu criar do zero um jogo interativo de Jo Ken Po.
Foi um grande desafio já que tive de adicionar algumas imagens de mapa geradas por I.A., diversas interações, sistema de vitória e derrota, pontuações dinâmicas, funções assíncronas, manipular eventos, manipular elementos do DOM, entre outros recursos utilizados.

Como neste ponto já tive amadurecimento em minha jornada dev, irei disponibilizar a pagina web hospedada pelo GitHub pages e o link dos arquivos no próprio GitHub.
